Landing page de un solo archivo que cataloga todos los Warframes del juego, con sistema de filtros por categoría, fondo biopunk/espacial en CSS puro y modales dinámicos de habilidades.

Fondo inmersivo Orokin — atmósfera biopunk/espacial construida con gradientes CSS en capas (sin imágenes externas): nebulosas, estrellas sintetizadas, grid de circuito y línea decorativa Orokin.

Catálogo completo — 40 Warframes organizados en 4 categorías: Ataque, Defensa/Tanque, Control y Soporte.

Filtros por categoría — navegación sticky con filtros animados que muestran/ocultan secciones con efecto fade-in.

Modal Maestro de Habilidades — un único modal Bootstrap reutilizable que se inyecta dinámicamente con los datos de cada Warframe. Arquitectura de alto rendimiento: cero modales ocultos en el DOM.

Base de datos JS — objeto warframeData extensible con nombre, descripción y número de cada habilidad.

Diseño premium dark — sistema de variables CSS con paleta Void (#09090b), acento energía (#38bdf8) y acento Orokin (#d4af37).

Responsive — grid de 4 columnas en escritorio, 2 en tablet, 1 en móvil. Modal adaptable.
